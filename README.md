### 增加路由器配置脚本
我编写了一个shell脚本，让它可以在Linux内核的路由器上被自动配置和更新，理论上也适用于任何Linux系统。仅使用小米路由R3进行测试。
使用下面的命令来配置它

        curl -O https://raw.githubusercontent.com/singleNeuron/Go-Hosts/master/go-hosts.sh -k
        chmod a+x go-hosts.sh
        ./go-hosts.sh
